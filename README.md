# CMP7005 Beijing Air Quality Prediction System

## Project Overview

This project was developed for CMP7005 Programming for Data Analysis.

The aim of the project is to analyse Beijing air quality data and build an interactive Streamlit application for predicting PM2.5 concentration using pollutant and meteorological variables.

## Project Tasks

The project follows the assessment rubric:

1. Task 1: Data Handling
2. Task 2: Exploratory Data Analysis
3. Task 3: Model Building
4. Task 4: Application Development
5. Task 5: Version Control

## Dataset

The project uses selected Beijing monitoring stations:

- Dongsi
- Guanyuan
- Shunyi
- Huairou
